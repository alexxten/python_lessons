2.Алгоритм по извлечению текста из таблицы
    1. Понять, как найти таблицу из html - найти тег и убедиться, что он уникален.
    Если не уникален - то можно найти все таблицы и взять нужную по индексу.
    2. Видим, что таблица состоит из tr - строк. Так как нам нужно распарсить всю таблицу,
    то можно просто пройтись циклом по ним.
    3. Одна tr состоит из нескольких td - ячеек. Из них и нужно взять текст.
    Опять можно использовать цикл for.
    4. Вытаскиваем текст из td.
    - 5. Занести текст в какую-то переменную.
    (подумать, в какой структуре удобнее хранить значения строки)

3. Запись датафрейма в csv, excel
